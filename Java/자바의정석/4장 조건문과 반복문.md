### 4-2

```java
public class Main {
    static public void main(String[] args) {
        int total = 0;
        for (int index = 1; index <= 20; index++) {
            if (index % 2 != 0 || index % 3 != 0) {
                total += index;
            }
        }
        System.out.println(total); //답: 174
    }
}
```
